# Iron — Workout Logger

> One sentence spec: "Log a set. Nothing else." Shipped in 5 days.

Live: [strongiron.vercel.app](https://strongiron.vercel.app)

---

## What it is

A minimalist workout logger built around a single job to be done. Constraints were the product.

## Product decisions

- **Scoped to one job ruthlessly** — if it wasn't logging a set, it didn't ship
- **Explicitly declined to build** social features, leaderboards, AI suggestions, and progress charts — each was a PM decision, not an oversight
- **5 days from spec to deployed** — speed was a byproduct of saying no

## Stack

React · TypeScript · Vite · Tailwind · Vercel
